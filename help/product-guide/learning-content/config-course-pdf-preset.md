# 設定PDF輸出預設集

建立預設集後，請設定PDF預設集設定。 預設集組態選項會整理在「一般」、「中繼資料」、「版面」、「安全性」、「列印」和「進階」標籤下。 如需詳細資訊，請檢視[PDF輸出預設集組態](../web-editor/native-pdf-web-editor.md)。

# 自訂編輯器 {#id181AI0003PN}

AEM Guides隨附強大的編輯器，可讓您的作者建立和編輯DITA檔案。 您可以自訂編輯器的工具列，以新增或移除您可以從工具列存取的任何功能。 此外，您也可以設定檔案的自動儲存、在元素插入檔案中時為其產生ID等等。

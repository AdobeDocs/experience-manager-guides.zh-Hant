# 設定On-Premise的新對應集合

下列步驟說明如何在內部部署環境中啟用新地圖集合。

1. 開啟Adobe Experience Manager Web主控台設定頁面。

   存取設定頁面的預設URL為：

   ```http
   http://<server name>:<port>/system/console/configMgr
   ```

1. 搜尋並選取&#x200B;**com.adobe.fmdita.config.ConfigManager**&#x200B;套件。

1. 啟用設定&#x200B;**啟用新的地圖集合** (enable.new.map.collections)。

   ![](assets/new-map-collection.png){width="800"}

1. 選取「**儲存**」。

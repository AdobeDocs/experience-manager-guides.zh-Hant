# 2026.01.0版中的已修正問題

本文介紹2026.01.0版Adobe Experience Manager Guides as a Cloud Service中不同區域修正的錯誤。

如需新功能和增強功能的詳細資訊，請參閱 [2026.01.0 版本中的新增功能](whats-new-2026-01-0.md)。

瞭解2026.01.0版[的](upgrade-instructions-2026-01-0.md)升級指示。

## 製作

- 使用內容選單中的編輯MathML選項更新內嵌MathML方程式時，更新後的值要等到頁面重新整理後才會反映出來。 (GUIDES-38198)
- 當主題包含許多在「可重複使用」面板中新增的可重複使用元素（具有ID的元素）時，由於容器高度固定，有些元素可能無法存取。 (GUIDES-37220)
- 插入檔案的互動參照時，地圖和主題的圖示相同。(GUIDES-36662)
- 編輯地圖時，`navtitle`中的特殊符號不會在Author檢視中顯示`topichead`。 (GUIDES-35435)
- 無法從&#x200B;**另存為新版本**&#x200B;對話方塊將多個版本標籤新增至主題。 (GUIDES-32716)

## 資產管理

- 無法從Assets UI的「版本記錄」面板中移除版本標籤。 (GUIDES-38276)
- 當上傳檔案名稱包含無效字元的資產時，該資產無法上傳並顯示不正確的訊息&#x200B;**檔案已由其他使用者**&#x200B;鎖定，儘管該資產已解鎖。 (GUIDES-32680)
- 在Assets搜尋中，子資產和中繼資料節點（例如影像和PDF）不正確地包含在結果中。 此外，對於套用DITAVAL篩選器時的輸出預設集，搜尋會傳回從條件預設集建立的內部產生的DITAVAL檔案。 (GUIDES-35418)

## 發佈

- 產生AEM Sites輸出時，包含關鍵字和主題標題及`<ph>`元素的對應標題不會包含在已發佈的輸出中。 (GUIDES-36641)
- 當使用包含連結至PDF但沒有範圍設定為外部的自訂預設集進行發佈時，該過程無法完成。 (GUIDES-21708)
- 執行大量啟動時，封裝建立會為列在頁面`fileReference`屬性下的所有路徑新增篩選器，包括外部和對等路徑。 (GUIDES-24887)
- 在原生PDF輸出中，`abbreviated-form`元素顯示`glossterm`而非指定的`glossSurfaceForm`或`glossAcronym`。 (GUIDES-26393)
- 對於原生PDF輸出，會忽略影像的`<alt>`元素，以防止替代文字套用至協助工具。 (GUIDES-29087)
- 在發行預設集期間，當下載具有基準的地圖的暫存檔案時，`metadata.xml`檔案錯誤地參照了`versionPath`而非`dampath`。(GUIDES-29815)
- 建立或編輯包含引文的主題時，如果未在引文對話方塊中新增「作者」欄位，則不會產生PDF。 (GUIDES-37934)
- CSS檔案(`rhdefault.css`)未正確套用至PDF範本，儘管未參考任何CSS，導致遺失CSS檔案錯誤記錄檔。(GUIDES-31752)

## Platform

- 嘗試儲存主題或地圖時，作業可能會間歇性地失敗，並出現&#x200B;**無法儲存檔案**&#x200B;錯誤，尤其是在密集資產處理工作或在背景執行的翻譯工作流程期間。 (GUIDES-37837)
- 使用`scope="external"`作為主題或地圖中DAM內容的參照會導致資產的相對路徑被GUID取代。 (GUIDES-38783)

## 報告

- 劃分清單報告錯誤地包含外部連結、有效的`keyrefs`和在目前對映範圍內已正確解析的關鍵字。 (GUIDES-27774)

## 已知問題

Adobe已找出下列2026.01.0版的已知問題：

- 從進行中的稽核任務中移除稽核中的主題時，其檔案狀態會繼續保持&#x200B;**在稽核中**，即使該主題不再是任何稽核任務的一部分。 (GUIDES-38709)<br>**因應措施**：從[屬性]頁面或[檔案屬性]面板，將主題的檔案狀態從[稽核中] **變更成適當的狀態**。
- 使用&#x200B;**尋找和取代**&#x200B;執行搜尋時，如果您從搜尋結果開啟檔案，關閉該檔案，然後嘗試透過選取列出的結果來重新開啟它，檔案將無法重新開啟。 (GUIDES-39050)<br>**因應措施**：先從搜尋結果開啟任何其他檔案，然後從清單重新開啟先前關閉的檔案，以解決問題。
- 搭配DB伺服器使用參考線時，對於包含自我參照的內容，主題清單報告會顯示每個自我參照的無效專案，導致檔案計數不正確。 (GUIDES-39420)

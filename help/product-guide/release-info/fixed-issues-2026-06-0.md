# 2026.06.0版中的已修正問題

本文介紹2026.06.0版Adobe Experience Manager Guides as a Cloud Service中多個區域修正的錯誤。

如需新功能和增強功能的詳細資訊，請檢視[2026.06.0版的新增功能](whats-new-2026-06-0.md)。

瞭解2026.06.0版[&#128279;](upgrade-instructions-2026-06-0.md)的升級指示。

## 製作

- 使用單位大小（例如`in`、`mm`或`px`）在影像屬性對話方塊中的&#x200B;**寬度**&#x200B;和&#x200B;**高度**&#x200B;欄位之間切換焦點時，值會遞增而非保持穩定。 (GUIDES-45929)

## 編輯器2.0

- 在`<keydef>`或`<topicref>`內的`<topicmeta>`中複製並貼上`<keywords>`會導致關鍵字被插入不需要的外部標籤中。 (GUIDES-45800)
- 以`mm`等單位指定的影像尺寸無法正確轉譯，導致影像以原始大小顯示，而非指定的尺寸。 (GUIDES-46275)
- 在啟用「標籤」檢視的情況下執行拖放時，選取內容連同部分XML或DITA標籤會留下不想要的孤立標籤，導致不正確的內容或檢視。 (GUIDES-28191)
- 在表格的「標籤」檢視中，當游標位於縮疊專案標籤正下方的儲存格時，按向上鍵可跳過縮疊的標籤，並將游標移至檔案的開頭。 (GUIDES-45408)
- 從表格內容工具列執行任何操作時，會意外關閉工具列，中斷後續的表格操作。 (GUIDES-45405)
- 從大綱檢視或階層連結使用&#x200B;**插入在後**&#x200B;或&#x200B;**插入在前**&#x200B;之後，游標會移至任意位置而不是新新增的標籤內。 (GUIDES-45147)
- 當&#x200B;**編輯MathML**&#x200B;選項以唯讀模式無法正確顯示，或檔案已由其他使用者取出時，該選項可讓使用者更新MathML內容，即使檔案不應為可編輯狀態。 (GUIDES-45172)
- 使用退格鍵刪除XML註解時，註解標籤在內容刪除後不會移除，且刪除作業會繼續到前一個元素。 (GUIDES-45240)
- 針對影像使用&#x200B;**複製路徑**&#x200B;或&#x200B;**複製UUID**&#x200B;選項，會傳回完整的轉譯路徑，而非原始資產路徑。 (GUIDES-45278)
- 將相同主題中的內容複製貼上到編輯器中的無效位置會插入非預期的`<foreign>`標籤。 (GUIDES-45378)
- 在Windows上，複製和貼上表格列會將非預期的屬性（例如`data-pm-slice`）新增至表格元素，造成無法儲存檔案的標示錯誤。 (GUIDES-45784)
- 使用地圖或主題中內容功能表的&#x200B;**複製**&#x200B;選項，然後貼上內容，在貼上的輸出中插入額外的`<data>`標籤。 (GUIDES-45703)
- 從`cmd`元素拖曳部分選取範圍時，內容無法在現有文字之間或另一個`cmd`元素的結尾拖曳。 (GUIDES-44883)
- 將`scale`屬性套用至資料表時，在製作和預覽模式中，資料表不會以設定的大小呈現。 (GUIDES-45984)
- 貼上從外部來源（如「油漆」或「剪裁工具」）複製的影像時，不會將影像插入主題。 (GUIDES-45983)
- 儲存後，從關鍵字對應衍生的主題標題中使用的`keyref`不會出現在目錄或主題標籤中，即使在瀏覽器重新整理後也是如此。 (GUIDES-45799)
- 使用註解版本的並排檢視在編輯器中開啟稽核任務，會將版本顯示為「無」，而不是主題的相關版本。 (GUIDES-45127)
- 存取「檢閱」任務頁面時，不會顯示主題之間的分頁，導致內容區段持續呈現。 (GUIDES-46777)
- 評論頁面樣式與新編輯器體驗不一致，導致不一致的視覺體驗(GUIDES-46061)

## 資產管理

- 當對應包含指向非DITA資源（例如`.html`）的外部`topicref`時，其預覽未顯示在Assets UI中。 (GUIDES-45409)
- 當選取大量來源資料夾時，「大量移動工具」對話方塊會溢位，且缺少卷軸，導致只有滑鼠的使用者無法存取目的地路徑欄位和動作按鈕。 (GUIDES-45805)
- 從Assets UI複製資產時，我們會收到`DXML reprocessing failure`的通知。 由於複製成功，此通知會造成誤導性並導致收件匣混亂。 (GUIDES-45012)
- 在資料夾的Assets左側欄中開啟「篩選器」面板時，搜尋欄位和Facet會維持停用狀態，直到面板關閉並重新開啟為止。 (GUIDES-42652)
- 排程的資產處理工作會無意中處理在翻譯工作流程期間建立的內容片段和中間翻譯資產，導致記錄中出現例外狀況，且對尚未準備就緒的資產進行錯誤處理。 (GUIDES-42582)

## 發佈

- 在PDF範本中使用`.plt`和`.css`檔案時，儘管不適用於這些檔案型別，但滑鼠右鍵內容功能表中仍會提供&#x200B;**產生ID**&#x200B;選項。 (GUIDES-45254)
- 在瀏覽器重新整理後重新開啟新建立的Native AEM Sites預設集時，**儲存**&#x200B;按鈕預設為未正確啟用，即使未進行任何變更亦然。 (GUIDES-45171)
- 複製標示為預設的原生PDF預設集時，也會將複製專案設為預設的預設集。 一次只能將一個預設集指定為預設集。 (GUIDES-44786)
- `outputclass`屬性不會傳播至產生的HTML或原生PDF輸出，以防止套用至MathML方程式的自訂類別影響轉譯的輸出。 (GUIDES-44393)
- 使用新AEM Sites範本產生的AEM Sites輸出大量啟用失敗，導致無法成功復寫至發佈執行個體。 (GUIDES-44049)
- 已識別DITA-OT套件中與AEM Guides搭配的易受攻擊的`jackson-databind`個JAR （2.9.8版）。 (GUIDES-43081)
- 從對應收集開啟任何原生AEM Sites輸出都會導致錯誤，指出找不到資源，導致無法存取產生的輸出。 (GUIDES-42065)
- DITA map中的`<reltable>`元素在產生Native PDF期間被忽略，導致輸出中缺少「Related concepts」、「Related tasks」和類似的自動產生互動參照區段。 (GUIDES-38333)
- 在AEM Sites上發佈具有`processing-role=resource-only`個元素的DITA map （具有舊版元件對應）時，會在其他情境（例如`topicgroup`元素和特定內容設定）中為這些元素產生孤立網站頁面。 (GUIDES-37650)
- 將長名稱的對應新增到對應集合時，對應集合UI會以扭曲的版面配置呈現。 此問題已透過新地圖集合解決。 (GUIDES-42062)
- 使用原生PDF引擎v1發佈：
   - 為特定內容產生原生PDF輸出時，儘管中間HTML包含多個頁面的完整內容，PDF中仍只會轉譯第一個頁面。 (GUIDES-28270)
   - 已啟用協助工具設定之原生PDF輸出內容的讀取順序不正確。 頁尾的頁碼會在主要內容之前讀取，而不是在結尾讀取。 (GUIDES-27790)
   - 原生PDF輸出中的色彩列不會延伸至整個頁面寬度，且在自訂頁面大小時重疊，導致某些色彩方塊隱藏。 (GUIDES-15505)
   - 原生PDF輸出中不遵循CSS `:is()`虛擬類別選擇器，導致與瀏覽器轉譯的樣式不同。 (GUIDES-11328)

  >[!NOTE]
  >
  > 上述問題已透過Native PDF引擎v2解決。 如需詳細資訊，請檢視[使用原生PDF引擎v2](../native-pdf/new-pdf-engine.md)。

## 翻譯

- 將基準線套用至包含許多資產的地圖，會延遲載入所選語言的翻譯報表，有時會在報表轉譯之前導致請求逾時。 (GUIDES-45508)
- 當選取的語言資料夾使用連字地區設定代碼（例如，`da-DK`）而不是底線格式(`da_dk`)時，Workspace設定底下「翻譯」索引標籤中的語言群組不會保留。 (GUIDES-37843)

## 檢閱

- 在主題名稱旁的檢閱UI左側面板中，**版本記錄**&#x200B;圖示的工具提示遺失。 (GUIDES-45511)
- 編輯作用中的稽核任務時，如果移除已屬於稽核一部分的某個主題，然後在不選取&#x200B;**更新**&#x200B;的情況下重新新增，則該主題的稽核者資訊會遺失。 (GUIDES-38774)
- 檢閱UI中的內容窗格在某些熒幕解析度上顯示水準卷軸，包括建議的1600px解析度，當內容延伸超過窗格寬度時，會導致隱藏檢閱評論。 (GUIDES-44082)

## 學習內容

- 使用「從問題庫插入」選項將問題新增至測驗時，儘管問題ID有效，簡答問題未列出。 (GUIDES-44942)
- 使用「從問題庫插入」選項插入問題時，「從問題庫插入」對話方塊會在開啟時意外閃爍或調整大小。 (GUIDES-45524)
- 使用從問題庫插入問題時，如果問題標題包含影像，則會發生錯誤。 (GUIDES-45383)
- 在修改範本標題時，選取SCORM輸出範本會導致應用程式錯誤。 (GUIDES-45521)
- 子標題(`.vtt`)檔案上傳到資料夾後，在存放庫或檔案總管檢視中不可見。 (GUIDES-46014)
- 在HTML預覽中載入視訊內容時，來源(`src`)路徑包含附加的轉譯路徑，而非保留原始檔案路徑，導致視訊無法正確轉譯。 (GUIDES-41776)
- 從Safari瀏覽器發佈SCORM輸出會導致套件下載為資料夾而非zip檔案，以及產生的內容中的轉譯和功能問題（例如，延伸內容、非作用中摺疊式功能表等）。 (GUIDES-45119)
- 在課程啟用「下一步」按鈕之前出現延遲，並影響學習者導覽體驗。 (GUIDES-45113)
- 儘管在預覽中正確顯示，但在發佈輸出時，簡短回答問題樣式呈現不正確。 (GUIDES-46478)
- 產生包含視訊的課程的PDF輸出時，不會轉譯視訊內容，且只會顯示檔案路徑，而非視訊縮圖或海報影像等預留位置影像。 此問題已透過在原生PDF輸出預設集中啟用&#x200B;**內嵌多媒體檔案**&#x200B;選項而解決。(GUIDES-45117)
- CSS虛擬類別和元素在指南的編輯器中以白色呈現，導致它們無法看到或難以在編輯器背景中讀取。 (GUIDES-45116)

## 已知問題

Adobe已找出下列2026.06.0版的已知問題：

## 編輯器2.0

- 在Source和作者模式之間切換會導致內容不一致，主題的部分消失或不會跨模式反映。 (GUIDES-47432)

- 使用啟用&#x200B;**追蹤變更**&#x200B;的大綱檢視時，拒絕變更會移除標籤內的整個內容，而非僅拒絕特定的修改內容。 (GUIDES-48319)

- 使用資料夾設定檔中的`editor_toolbar.json`自訂編輯器工具列時，預覽模式中的&#x200B;**匯出為PDF**&#x200B;按鈕不會執行任何動作。 (GUIDES-47525)

- 執行刪除操作時，可能會在影像地圖、結構化元素、內嵌格式標籤及不可合併的區塊之間發生游標移動和導覽的一些細微不一致，有時會導致意外的游標或刪除行為。 (GUIDES-46756)

- 在地圖編輯器的中斷連結上使用`Ctrl+click`會觸發應用程式錯誤。 (GUIDES-45544)

- 在唯讀內容（例如`conref`段落）之後緊鄰的段落開頭按退格鍵，可能會意外刪除或合併可編輯的段落，導致意外刪除可編輯的段落。 (GUIDES-45049)

- 將條件指標套用至`bodydiv`之類的元素時，指標會溢滿至完整標籤檢視中的相鄰標籤，導致不正確的視覺呈現。 (GUIDES-44971)

- 使用編輯器時，您無法選取`keyrefs`或衍生自對映的索引鍵（包括在`term`或`abbreviated-form`元素中參考之對映內的字彙專案），導致編寫體驗降低。 (GUIDES-45790) <br> **因應措施**：您可以使用屬性值，從右側面板變更`keyref`的值。

- 在「大綱」檢視中，關閉並重新開啟主題後重新啟用&#x200B;**顯示文字**&#x200B;並不會在元素標籤旁邊顯示文字。 (GUIDES-48320) <br> **因應措施**：啟用&#x200B;**顯示文字**&#x200B;選項，然後重新開啟主題。 重新開啟後，文字會正確顯示在元素標籤旁邊。

- 當使用&#x200B;**重新命名元素**&#x200B;選項重新命名內嵌標籤時，階層連結不會立即更新，而且只有在游標移至標籤中或檢視模式變更後，才會反映變更。 (GUIDES-44993) <br> **因應措施**：重新命名內嵌標籤後，請重新整理瀏覽器以更新階層連結。

- 將MathML方程式插入為`conref`時，無法正確轉譯。 (GUIDES-46601) <br> **因應措施**：將MathML方程式包裝在`<p>`元素內，並將該`<p>`元素用作conref來源。

## 檢閱

- 當稽核任務重新指派給專案團隊之外的使用者時，儘管缺少專案成員資格，使用者可以執行稽核動作，而稽核者可視性、稽核狀態追蹤和委派通知無法正常運作。 (GUIDES-46602)

## 發佈

- 當使用透過條件設定排除所有專案符號清單專案的DITAVAL篩選發佈內容時，輸出錯誤地保留空白專案符號標籤，而不是移除整個清單結構。 (GUIDES-47238)

- 以新基準發佈原生AEM網站輸出時，「主題清單」會呈現空白而無法顯示所選基準中包含的主題。 (GUIDES-46480) <br> **因應措施**：使用可透過設定管理員設定的舊基準線發佈原生AEM網站輸出。

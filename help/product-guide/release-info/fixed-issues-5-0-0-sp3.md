# 已修正5.0.0 Service Pack 3版本（2026年2月）中的問題


本文章說明5.0.0 Service Pack 3版本Adobe Experience Manager Guides中不同區域所修正的錯誤。

瞭解5.0.0 Service Pack 3版本[&#128279;](upgrade-instructions-5-0-0-sp3.md)的升級指示。

## 翻譯

- 當最初以特定版本（例如，在`/en/`下）管理為語言特定資產的影像移至具有更新版本的全域資料夾並執行基準匯出時，新基準會繼續參照該影像的過時語言特定版本，導致基準匯出失敗。 (GUIDES-39394)

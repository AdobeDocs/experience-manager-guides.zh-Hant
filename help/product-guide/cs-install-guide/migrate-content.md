# 移轉現有內容 {#id181AGB000Y4}

AEM Guides可讓您將各種結構化和非結構化檔案轉換為DITA格式。 本主題涵蓋有關在AEM存放庫中上傳DITA內容以及將非DITA內容轉換為DITA格式的資訊。

>
> 請參閱 *內容移轉與上傳* 最佳作法指南中的區段，以瞭解使用移轉及上傳現有內容的最佳作法。

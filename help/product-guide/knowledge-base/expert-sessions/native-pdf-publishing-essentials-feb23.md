# 原生PDF發佈要點

AEM Guides內建的PDF發佈引擎讓使用者能以PDF格式建立、開發和發佈內容。 其豐富的功能集可建立多樣化的頁面配置、自訂樣式表，以及設計無縫內容發佈所需的各種PDF元件。

## 工作階段概述

在這場會議中，您將瞭解如何使用AEM Guides中的原生PDF引擎發佈PDF

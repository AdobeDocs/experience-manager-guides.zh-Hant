# 工作階段概述

定義清晰的網頁體驗版面配置可產生一致的應用程式品牌
與設計，以及整個數位環境的統一性與一致性
體驗/頁面數量。
所有這些都可透過Adobe Experience Manager Guides提供的AEM網站輸出的不同變化來實現。
在此課程中，您將瞭解如何產生AEM網站輸出的不同變體。

## 關鍵重點

在本次研討會中，您將瞭解下列內容：

- AEM中的不同範本型別產品
- 瞭解如何根據範本或發佈機制建立變體
- 使用每個範本/機制的差異
- 使用既有的Sites範本來建置AEM Sites輸出

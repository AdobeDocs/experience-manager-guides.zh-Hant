# 使用氧氣 {#id221BH0V0UE8}

Oxygon XML Author提供使用者易記的編寫功能，並提供簡單易用的介面。

您可以使用相同的來源產生PDF、ePub、HTML和許多其他格式的輸出。

*請聯絡您的客戶成功團隊，讓此功能在環境中啟用。 這並非是現成支援的一部分。*

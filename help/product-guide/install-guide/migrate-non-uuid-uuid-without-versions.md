# 移轉不含版本的非UUID內容

>[!IMPORTANT]
>
> 如果您打算略過或不移轉資產版本，可以選擇此移轉方法。


1. 使用AEM案頭應用程式等Adobe工具，直接從AEM Assets UI從非UUID執行個體下載資產並上傳至UUID執行個體。

1. 確保您啟用DAM更新資產工作流程，並在匯入內容以建立GUID後在所有資產上執行該工作流程。
